# Menu projekt v React
Tento projekt je inspirovaný od FreeCodeCamp, s tím že je vytvořený v React a je to moje "recreation" již existujícího projektu, co jsem dělal v minulosti.

# Funce stránky
Tento projekt obsahuje základní funkce skoro každé moderní stránky, a sice filtrovat určitý seznam jak podle typu, tak i podle ceny
- Možnost filtrovat jídla podle jejího typu (snídaně, večeře, shaky...)
- Možnost filtrovat taktéž podle jejich ceny
